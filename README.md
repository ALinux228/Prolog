## Задание 1
### Рекурсия
Составить программу, определяющую, в каком из данных двух чисел больше цифр.
### Алгоритм решения
Подсчет количества цифр (предикат count_digits):
  1. Базовый случай: число 0 содержит 1 цифру
  2. Рекурсивный случай для положительных чисел: делим число на 10, рекурсивно подсчитываем цифры в частном, затем добавляем 1
  3. Для отрицательных чисел: берем модуль числа и считаем цифры
Сравнение (предикат more_digits):
  1. Вычисляем количество цифр для обоих чисел
  2. Сравниваем результаты:
  3. Если в первом больше, то возвращаем first
  4. Если во втором больше, то возвращаем second
  5. Если равны, то возвращаем equal

## Задание 2
### Списки
Найти сумму тех элементов списка, которые оканчиваются на заданную цифру
### Алгоритм решения
Определение последней цифры (предикат last_digit):
  1. Берем абсолютное значение числа
  2. Вычисляем остаток от деления на 10
  3. Рекурсивный обход списка (предикат sum_ending_with):
    Базовый случай: для пустого списка сумма равна 0
    Рекурсивный случай: проверяем последнюю цифру головы списка
  4. Если совпадает с заданной, то добавляем элемент к сумме и продолжаем рекурсию
  5. Если не совпадает, то просто продолжаем рекурсию

## Задание 3
### Решить задачу на обработку списка
Определим множество как список без повторяющихся элементов. Найти объединение множеств.
### Алгоритм решения
Проверка принадлежности элемента списку (предикат member_of):
  1.Базовый случай: элемент является головой списка
  2.Рекурсивный случай: элемент принадлежит хвосту списка
Удаление дубликатов (предикат remove_duplicates):
  1. Базовый случай: пустой список не содержит дубликатов
  2. Рекурсивный случай:
    2.1. Если голова встречается в хвосте - пропускаем её
    2.2. Если не встречается - включаем в результат
Объединение множеств (предикат union_sets):
  1. Объединяем два списка в один
  2. Удаляем дубликаты

## Задание 4
### Решить логическую задачу
Три подразделения А, В, С торговой фирмы стремились получить по итогам года максимальную прибыль. Экономисты высказали следующие предположения:
1) А получит максимальную прибыль только тогда, когда получат максимальную прибыль В и С,
2) Либо А и С получат максимальную прибыль одновременно, либо одновременно не получат,
3) Для того, чтобы подразделение С получило максимальную прибыль, необходимо, чтобы и В получило максимальную прибыль.
По завершении года оказалось, что одно из трех предположений ложно, а остальные два истинны. Какие из названных подразделений получили максимальную прибыль?
### Алгоритм решения
Обозначения: A, B, C – логические переменные (true – получило прибыль, false – не получило)

Формализация условий:

Условие 1: из A следует (B и C)

Условие 2: A эквивалентно C

Условие 3: из C следует B

Метод решения:
1. Перебираем все 8 возможных комбинаций (2^3)
2. Для каждой комбинации вычисляем истинность каждого условия
3. Находим комбинации, где сумма истинных условий равна 2 (одно ложно, два истинны)
